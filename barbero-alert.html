📱 Come installare sul telefono

1Carica questo file barbero-alert.html su un repo GitHub e attiva GitHub Pages (Settings → Pages → branch main).
2Android (Chrome): apri la pagina, premi "Attiva Notifiche" e accetta il permesso. Fatto!
3iPhone (Safari ≥ iOS 16.4): premi il tasto Condividi ↑ → "Aggiungi a schermata Home" → apri l'app → "Attiva Notifiche".
4Tieni la pagina aperta in background oppure aggiungila alla schermata home: il sistema controlla i nuovi eventi ogni ora automaticamente.
5Ogni 24 ore il sistema usa l'AI per verificare i siti di riferimento e aggiornare automaticamente la lista eventi senza che tu debba fare nulla.
